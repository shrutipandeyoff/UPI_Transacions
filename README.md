# UPI Transaction Analysis Dashboard | Power BI

## Overview

Developed an interactive Power BI dashboard to analyze UPI transaction trends, user behavior, and payment performance. Built as part of a hands-on Data Analytics Bootcamp project to transform transactional data into actionable business insights using data modeling, Power Query, and DAX.

---

## Business Objectives

* Analyze transaction volume and value.
* Monitor payment trends and user behavior.
* Identify high-performing categories and channels.
* Enable interactive reporting through dynamic filters.

---

## Features

* KPI Dashboard
* Transaction Trend Analysis
* User & Category Analysis
* Interactive Slicers
